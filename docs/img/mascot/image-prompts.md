# Psy the Owl — AI Image Prompts

Each prompt is fully self-contained. Copy the entire block into your AI image
generator (ChatGPT/DALL-E, Midjourney, Stable Diffusion, etc.).

Recommended generation settings:
- Resolution: 1024×1024 px
- Format: PNG with transparent background
- After generating, run the padding trimmer (see README note at bottom)

---

```
Style: flat cartoon, modern vector, clean outlines, no gradients,

Psy stands upright in a relaxed, neutral pose facing the viewer directly,
with a calm and friendly closed-mouth smile. Wings rest naturally at the

Please generate a new png image now with a fully transparent background.
```

## Trimming Padding After Generation

AI generators often add excess transparent padding. After saving images,
trim them with:

```bash
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
```
